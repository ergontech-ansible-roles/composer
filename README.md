Composer Role
=========

Installs and configures [Composer](https://getcomposer.org/)

Dependencies
------------

```
- php
- curl
- git
- unzip
```

Role Variables
--------------

```
github_oauth_token ( not required )
composer_version: default(1.6.4)
composer_use_latest: default(false)
```
